# Reviews Upgrade Patch
- Полные отзывы: кнопка «Читать полностью/Свернуть».
- Кнопка «Написать отзыв»: модалка с формой (имя, питомец, оценка, текст, согласие).
- Сохранение локально (для мгновенного UX) + отправка на `/api/reviews` (заглушка).
- Средняя оценка и количество, звёзды, JSON-LD AggregateRating.
- Фирменная стилистика OnlyVet (Navy/Teal/Coral, большие радиусы, спокойные тексты).
