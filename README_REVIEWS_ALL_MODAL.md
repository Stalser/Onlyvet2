OnlyVet — Reviews block (limited + All reviews modal)
====================================================

Что сделано:
- На странице видно первые N отзывов (сейчас 6).
- Кнопка «Все отзывы (X)»: если отзывов >99, показывается «99+».
- При клике — открывается отдельное окно (модалка) со всеми отзывами (прокрутка, кнопка закрытия).
- «Читать полностью» на карточке главного блока открывает модалку полного отзыва.
- «Написать отзыв» — ваш существующий ReviewModal.

Файлы:
- components/Reviews.tsx
- components/AllReviewsModal.tsx

Как применить:
1) Замените у себя components/Reviews.tsx этим файлом.
2) Добавьте новый components/AllReviewsModal.tsx.
3) Commit → дождитесь сборки на Vercel.

Примечание:
- Если у вас компонент формы называется иначе (не ReviewModal), поправьте импорт в Reviews.tsx.
