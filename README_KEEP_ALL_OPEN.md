OnlyVet — Reviews: keep All Reviews open when reading a full review
===================================================================

Что сделано:
- При клике «Все отзывы (X)» открывается модалка со всеми отзывами.
- При клике «Читать полностью» открывается второе окно поверх (full-view).
- При закрытии полного отзыва окно «Все отзывы» остаётся открытым (НЕ закрывается).
- Поддержка фотографий: один `photo` или массив `photos` (рендерим галерею в полном окне).

Файлы:
- components/Reviews.tsx            — проводка логики (не закрываем showAll при открытии полного отзыва)
- components/AllReviewsModal.tsx    — список всех отзывов (с фото-обложкой, если есть)
- components/ReviewFullModal.tsx    — модалка полного отзыва (галерея, стрелки, ESC, без history.back())

Интеграция:
1) Замените ваши файлы в components/ на указанные версии.
2) Если в API ответы содержат `photos: string[]`, галерея заработает автоматически. Если только `photo`, тоже работает.
3) Коммит → Vercel пересоберёт сайт.
